# TimeTracker – Überstunden Tracker

A simple overtime tracking application built with React + Vite. Track your weekly working hours, manage comp time (Stundenabbau), and see your accumulated overtime balance.

## Features
- Weekly hours tracking (41h normal / 40h holiday week target)
- Accumulated overtime balance display
- Hours reduction (Stundenabbau) support
- Persistent storage via localStorage
- German language UI

## Development
```bash
npm install
npm run dev
npm run test
npm run build
```
